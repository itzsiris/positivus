# Positivus - Beautiful Landing Page

A modern, responsive landing page template built with **Sass**.

## Preview

![Preview](./images/preview.png)

## Features

* Modern and clean design
* Fully responsive layout  
* Built with **Sass** preprocessor
* Easy to customize with Sass variables
* [Figma template](https://www.figma.com/community/file/1230604708032389430)

## Getting Started


1. Install Node.js
Download and install Node.js from [nodejs.org](https://nodejs.org/)

2. Clone the repository:

```bash
git clone https://github.com/itzsiris/positivus
cd positivus
```

3. Initialize npm project

```bash
npm init
```

4. Install Sass

```bash
npm install sass --save-dev
```

5. Add script to package.json
```bash
  "sass-watch": "sass --watch ."
```

6. Run Sass watch
```bash
  npm run sass-watch
```

7. Open index.html in your browser.
