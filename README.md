# Positivus - Beautiful Landing Page

A modern, responsive landing page template built with **Sass**.

## Preview

![Preview](./images/preview.png)

## Features

* Modern and clean design
* Fully responsive layout  
* Built with **Sass** preprocessor
* Easy to customize with Sass variables
* [Figma template](https://www.figma.com/community/file/1230604708032389430/positivus-landing-page-design)

## Getting Started


1. Clone the repository:

```bash
git clone https://github.com/itzsiris/positivus
cd positivus
```

2. Initialize npm project

```bash
npm init
```

3. Install Sass

```bash
npm install sass --save-dev
```

4. Add script to package.json
```bash
  "sass-watch": "sass --watch ."
```

5. Run Sass watch
```bash
  npm run sass-watch
```

6. Open index.html in your browser.
